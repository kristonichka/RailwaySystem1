Лабораторная работа №5: Приложение для управления SQL-базой данных

Описание проекта:

Десктопное приложение для управления базой данных железнодорожного вокзала, разработанное на C++/CLI с использованием Windows Forms и SQL Server. 
Приложение предоставляет графический интерфейс для выполнения CRUD-операций, выполнения SQL-запросов и управления данными.

Функциональные возможности
1. Подключение к SQL-серверу с настройкой параметров подключения
2 Полный цикл CRUD-операций (добавление, чтение, обновление, удаление)
3. Выполнение SQL-запросов произвольной сложности
4. Фильтрация и сортировка данных в реальном времени
5. Импорт/экспорт данных в форматах CSV, XML, JSON
6. Валидация данных на стороне клиента и сервера
7. Логирование операций с базой данных
8. Защита от SQL-инъекций через параметризованные запросы

Технологический стек:
1. Язык программирования: C++/CLI (.NET)
2. Графический интерфейс: Windows Forms
3. База данных: Microsoft SQL Server / SQLite
4. Среда разработки: Visual Studio 2022
5. Система контроля версий: Git
